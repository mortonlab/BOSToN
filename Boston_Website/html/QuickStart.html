BOSToN

Standalone Windows Version

Quick Start Instructions

I.                    Download BOSToN (a 30-day trial copy can be available by special request)

II.                  Extract to C:\Program Files (if extracted elsewhere, you may break the links)

III.                Download and install Python 2.7.6 (https://www.python.org/)

IV.                Download and install NumPy

a.      For 32 bit use numpy-MKL-1.8.1.win32-py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/

b.      For 64 bit use numpy‑MKL‑1.8.1.win‑amd64‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/

V.                  Download and install BioPython

a.      For 32 bit use biopython‑1.63.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/

b.      For 64 bit use biopython‑1.63.win‑amd64‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/

VI.                Double left click on BOSToN icon (in C:\Program Files\BOSToN)

VII.              Enter location and name of FASTA txt file (for an example case enter: ex_pp.txt)
